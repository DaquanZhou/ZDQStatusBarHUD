# ZDQStatusBarHUD
状态栏提示信息

拉取项目后只需要将ZDQStatusBarHUD拷贝至自己项目中即可使用

1.成功提示

    [ZDQStatusBarHUD showSuccess:@"成功哦！"];
    
2.失败提示

    [ZDQStatusBarHUD showError:@"失败啦！"];
    
3.loading提示

    [ZDQStatusBarHUD showLoading:@"进行中的啦..."];
    
4.普通提示

    [ZDQStatusBarHUD showText:@"随便显示的文字！！！！"];
    
5.提示消失

    [ZDQStatusBarHUD hide];
